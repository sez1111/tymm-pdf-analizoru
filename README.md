# TYMM PDF Analizörü - Kolay Başlatma Paketi

## Windows

1. ZIP'i çıkar.
2. `BASLAT_WINDOWS.bat` dosyasına çift tıkla.
3. Tarayıcı açılırsa uygulama hazırdır.
4. Açılmazsa tarayıcıda şu adresi aç: http://localhost:8501

## Mac

1. ZIP'i çıkar.
2. Terminal aç.
3. Klasörün içine gir.
4. Şunu çalıştır:

```bash
bash BASLAT_MAC.command
```

## Not

İlk çalıştırmada gerekli paketler kurulur. İnternet gerekir.
